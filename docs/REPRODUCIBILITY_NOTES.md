# Reproducibility Notes

The included materials support review of a repaired-unified high-H0 narrow posterior branch. They are not yet a complete one-command fresh rerun package.

Included:

- Cobaya YAML for the repaired-unified high-H0 narrow rerun;
- summarized chain audit outputs;
- posterior summary CSV;
- per-chain summary CSV;
- scripts for local asset checks from the uploaded reproducibility package.

Omitted from GitHub-ready folder:

- full chain text files;
- local-only Planck likelihood archive;
- unreviewed full CLASS modified source overlay.

A future full reproducibility release should either provide a license-clean patch/diff against a specified CLASS version or clear instructions for obtaining and patching CLASS locally.
