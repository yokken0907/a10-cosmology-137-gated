# Technical Visual Orientation Scope

## Intended audience

This page is meant for:

- technically interested first-time readers,
- Jxiv readers who follow the repository link,
- researchers or engineers who want a fast orientation before opening the detailed files,
- readers who understand that a visual overview is not a validation result.

It is not optimized for children, casual browsing, or total beginners.

## Function

The page provides a compact orientation to:

1. reader-level assumptions,
2. localized early-time transfer near `z≈5000`,
3. stability-gated activation near `χ≈137`,
4. repository workflow and provenance chain,
5. observable logic,
6. evidence hierarchy,
7. recommended reading order,
8. explicit non-claims.

## Non-function

The page does not execute or validate:

- CLASS,
- Cobaya,
- MCMC,
- Planck/clik likelihoods,
- BAO likelihoods,
- SN likelihoods,
- SH0ES likelihoods.

## Claim boundary

The page must not be cited as numerical evidence. Its purpose is orientation and communication only.

Correct interpretation:

```text
This page helps readers understand where the evidence is and what the model is trying to do.
```

Incorrect interpretation:

```text
This page proves the A10 137-gated branch.
```
