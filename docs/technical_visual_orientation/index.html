0. Reader level and purpose

This page assumes the reader can tolerate terms such as H0, CMB, sound horizon, CLASS, Cobaya, and MCMC. It does not try to teach cosmology from zero.

Its role is to reduce the first-contact friction of the repository: before reading source overlays, YAML files, claim-boundary notes, and MCMC summaries, the reader can see the intended mechanism and evidence chain in one place.

Recommended placement:
Use this as an optional repository page under docs/technical_visual_orientation/. It should sit after the README and before deeper reproducibility materials.

The model is not useful if it raises H0 while catastrophically degrading CMB acoustic structure or BAO distances. The central visual idea is therefore not "large change", but "localized change with controlled residuals."

Technical reading

  • TT residuals must not become a broad uncontrolled phase distortion.
  • EE/TE should remain nontrivial tests, not be hidden by the explanation.
  • BAO distance consistency matters because early-time success can fail at late times.
  • Full model comparison requires likelihood-level comparison, not this page.

5. How to read the repository first

For a technically interested reader, the fastest route is not to open every file. Start from the claim boundary and provenance map, then move to configuration and source materials.

Suggested reading order

  3. docs/REPRODUCIBILITY_NOTES.md
  4. class_patch/

6. Claim boundary: the page should not expand the science

Allowed reading

  • A structured candidate branch for high-H0 cosmology.
  • A repository-facing guide to the model and archive layout.
  • A conceptual explanation of z-localization and χ-gating.
  • A pointer toward the actual manuscript and reproducibility materials.

Forbidden reading

  • "A10 has solved the Hubble tension."
  • "137 is proven to be a universal cosmological constant."
  • "This HTML is a numerical validation."
  • "The repository is a full one-click independent reproduction."
  • "Visual clarity replaces likelihood comparison."

The goal is a professional orientation page: visually helpful, technically honest, and restrained.
