# Data and Likelihood Requirements

This public repository does not redistribute Planck likelihood files, clik assets, or other external likelihood data.

A third-party rerun requires:

- compatible CLASS checkout/build environment;
- Cobaya installation;
- Planck 2018 likelihoods and clik configured locally;
- BAO/SN/SH0ES likelihoods matching the manuscript workflow;
- local paths adapted in the Cobaya YAML.

The source archive contained a local-only Planck minimal archive. That asset is intentionally omitted from this GitHub-ready folder.
