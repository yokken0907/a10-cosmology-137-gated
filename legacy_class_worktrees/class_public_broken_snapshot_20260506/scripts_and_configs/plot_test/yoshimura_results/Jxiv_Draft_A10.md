# Title: A Viable Early-time Solution to the Hubble Tension via Decaying Dark Sector at z ~ 5000

## Author: Yoshi (Independent Researcher)
## Date: 2026-03-12

### Abstract
We present a self-consistent cosmological model to mitigate the Hubble tension. By implementing a decaying two-fluid sector (Quasi-vacuum to Stiff-fluid) at the pre-recombination epoch (z ~ 5000) within the CLASS framework, we demonstrate that H0 = 73 km/s/Mpc can be achieved while maintaining standard values for n_s and N_eff. Linear perturbation analysis shows excellent agreement with the Planck-like CMB power spectrum.
