# Limitations

## Scientific limitations

- The model is a candidate high-H0 mechanism, not a definitive resolution.
- Full model selection over all cosmological alternatives has not been completed.
- The repaired-unified branch posterior is narrow by construction.
- The trigger position is not fully marginalized in a broad free-gate posterior.
- Matched baseline workflows are needed before making strong final superiority claims.
- Root-geometric material is conceptual support, not literal cosmological derivation.
- The optional late-time drag sector is not the core H0 mechanism.

## Reproducibility limitations

- The exact upstream CLASS commit remains unreconstructed.
- The repository provides a local modified source overlay, not a clean upstream patch release.
- Planck/clik and other external likelihood data are not redistributed.
- Full raw chain files are not included in the GitHub body.
- Reproducing the full workflow requires a compatible local CLASS/Cobaya/likelihood environment.

## Publication limitations

- This is AI-assisted independent research.
- Independent expert review is required.
- Public release should not be interpreted as observational model selection.
