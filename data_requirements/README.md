# Data requirements

The original local folder included Planck/BAO/SN likelihood data under `packages/data/`. These are not redistributed here.

A fresh reproduction requires the user to install the appropriate Cobaya likelihood packages and place external data in the expected local paths. The included `planck_2018_minimal_manifest_local_reference.sha256` is only a local reference manifest, not a redistributed data package.

Public repository policy:

- document external data requirements
- do not redistribute third-party likelihood bundles unless their license explicitly permits it
- keep local-only data outside Git
